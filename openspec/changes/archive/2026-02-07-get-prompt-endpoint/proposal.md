## Why

Telegram бот и другие клиенты должны иметь возможность получить текущий системный промпт агента для отображения или отладки. Сейчас промпт загружается из Vertex AI Prompt Management, но нет способа узнать его содержимое через API.

## What Changes

- Добавить эндпоинт `GET /api/prompt` для получения текущего системного промпта
- Возвращать текст промпта и его длину

## Capabilities

### New Capabilities

- `get-prompt`: Эндпоинт для получения текущего системного промпта агента

### Modified Capabilities

(нет)

## Impact

- `app.py`: Новый эндпоинт `/api/prompt`
- API: Новый GET эндпоинт для чтения промпта
