## Why

Системный промпт загружается из Vertex AI Prompt Management только при холодном старте контейнера. Чтобы применить изменения промпта, требуется редеплой сервиса. Нужна возможность перезагружать промпт без редеплоя — по запросу из Telegram бота.

## What Changes

- Добавить эндпоинт `POST /api/reload-prompt` для перезагрузки системного промпта
- Хранить текущий промпт в `app.state` для возможности обновления
- Пересоздавать агента с новым промптом при вызове эндпоинта

## Capabilities

### New Capabilities

- `prompt-reload`: Эндпоинт для перезагрузки системного промпта из Vertex AI без редеплоя

### Modified Capabilities

- `prompt-management`: Добавить поддержку динамического обновления промпта в runtime

## Impact

- `app.py`: Новый эндпоинт `/api/reload-prompt`, изменение хранения агента в state
- `agent/adk_agent.py`: Возможно рефакторинг для поддержки пересоздания агента
- API: Новый эндпоинт, вызываемый из Telegram бота
