## Context

Системный промпт загружается из Vertex AI Prompt Management при старте и может быть перезагружен через `/api/reload-prompt`. Текущий промпт хранится в агенте (`app.state.agent.instruction`). Нужен эндпоинт для получения этого промпта.

## Goals / Non-Goals

**Goals:**
- GET эндпоинт для получения текущего системного промпта
- Возвращать текст промпта и метаданные (длина)

**Non-Goals:**
- Изменение промпта через этот эндпоинт (для этого есть reload-prompt)
- Авторизация (Cloud Run IAM)

## Decisions

### 1. Получение промпта из агента

**Решение**: Читать `app.state.agent.instruction` напрямую.

**Обоснование**: Агент уже хранится в state, instruction — публичный атрибут.

### 2. Формат ответа

**Решение**: Возвращать `{"prompt": "<text>", "length": <int>}`.

**Обоснование**: Минимальный полезный ответ, совместимый со стилем других эндпоинтов.

## Risks / Trade-offs

**[Промпт может содержать конфиденциальную информацию]** → Эндпоинт защищён Cloud Run IAM, как и остальные.
